# Task Manager — Challenge Starter Project

Use this project only after Slide 27, during the 60-minute Copilot challenge.

## What's already built

- View a list of tasks
- Add new tasks
- Delete tasks
- Mark tasks as Done / Pending

## How to run it

```bash
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.

## Your challenge

Open the root **CHALLENGE_GUIDE.md** and **COPILOT_CHEATSHEET.md** to see the scored features and prompts.

Do not use this folder for Slides 10-26. Use `Workshop_Buildalong_Starter/` for the teaching section.

## File structure

```
src/
├── App.jsx              ← Main app — start here
├── App.css              ← All styles (you can modify these)
├── main.jsx             ← Entry point (don't touch this)
├── components/
│   ├── Header.jsx       ← Top bar with task count
│   ├── TaskCard.jsx     ← Individual task card
│   └── AddTaskForm.jsx  ← Input and Add button
└── data/
    └── sampleTasks.js   ← Pre-loaded tasks
```
