# Task Manager — Workshop Starter Project

Welcome to the workshop! This is your starting point.

## What's already built

- View a list of tasks
- Add new tasks
- Delete tasks
- Mark tasks as Done / Pending

## How to run it

```bash
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.

## Your challenge

Open **CHALLENGE_GUIDE.md** to see what features you'll be building today using React and GitHub Copilot.

## File structure

```
src/
├── App.jsx              ← Main app — start here
├── App.css              ← All styles (you can modify these)
├── main.jsx             ← Entry point (don't touch this)
├── components/
│   ├── Header.jsx       ← Top bar with task count
│   ├── TaskCard.jsx     ← Individual task card
│   └── AddTaskForm.jsx  ← Input and Add button
└── data/
    └── sampleTasks.js   ← Pre-loaded tasks
```

Good luck — and remember, GitHub Copilot is here to help! 🚀
